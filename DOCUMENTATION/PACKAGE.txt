=================
Versioning Scheme
=================

vX.X.X-ux

Required Changes
- zor/DEBIAN/control
- zor/DEBIAN/postinst

[WITH DATES AND VERSIONS!]

- zwn/DEBIAN/control
- zwn/usr/share/zyphor-whats-new/release-notes.html
- zcc/DEBIAN/control
- zcc/usr/share/zyphor-command-center/ui/home.py

[END OF WITH DATES AND VERSIONS!]

- updates/DEBIAN/control (if new default package was introduced).

=================================================
List of Packages Develop Under the Zyphor Project
=================================================

- zcli               (zyphor-cli)
    CLI Abstraction Layer.

- zcc    (zyphor-command-center)
    CLI Abstraction Layer GUI, runs on top of bash or zsh.

- zor        (zyphor-os-release)
    OS release file located in /usr/lib/os-release -> /etc/os-release.

- zou        (zyphor-os-updates)
    Small OS updates (new feature, patch release and others...)
    * fastfetch-config-1
    * grub-screensaver-1
    * zyphor-display-mac-v1

- zyshell             (zyshell)
    Zyphor OS default shell. Pre-installed and pre-configured after new installed OS, update and upgrade.

- zysh             (zysh)
    Replacement for zyshell.

- updates           (zyphor-updates)
    Dependencies, default applications and configurations in every new release.

- zwn         (zyphor-whats-new)
    A lightweight release notes viewer for Zyphor OS that shows system updates
    and changes in the browser after upgrades.

- zrc         (zyphor-repo-config)
    Zyphor OS repository configuration
    Installs and manages the Zyphor OS APT repository configuration.

- wallpapers
    - zyphor-wallpapers-default-2026-2027
    - zyphor-wallpapers-nature
    - zyphor-wallpapers-pragmata
